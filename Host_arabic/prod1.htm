Oracle Cloud Infrastructur تأجير الحوسبة السحابية
السعر الشهري المؤلف/ الناشر
يورو120 شركة أوراكل Oracle Corporation

تستطيع أيضاً استئجار الحوسبة السحابية عبر منتجرنا " APEX Store Premium " في خدمة Oracle Cloud Instruction لشركة Oracle Corporation. ودون الحاجة لتسجيل حساب على موقع Oracle.com.

باستخدام زر "زائد" أو "ناقص"، يمكنك ضبط مبلغ الاشتراك الشهري، والذي يبلغ إجماليه 0,120 يورو مقابل 20 جيجابايت من مساحة القرص.

المكونات الإضافية، مثل قاعدة بيانات Oracle Enterprise 23ai أو النظام الأساسي APEX، مجانية تمامًا. ولزيادة المساحة بمقدار 20 جيجابايت أخرى، ما عليك سوى دفع مبلغ مضاعف للاشتراك الأساسي.

يعد هذا حلاَ جيداً للعميل الذي بدأ في اختبار الحلول السحابية حديثاً. ومع ذلك، يوصي فريقنا باستخدام Oracle Cloud Infrastructure، مباشرة على موقع Oracle.com.
